Created Custom module called it mymodule

1. A new form text field named "Site API Key"  adds to the "Basic Site Information" form with 
the default value of “No API Key yet”.

2.When this form is submitted, the value that the user entered for this field is saved as the system variable 
named "siteapikey".

3.A Drupal message informs the user that the Site API Key has been saved with that value.


4. When this form is visited after the "Site API Key" is saved, the is populated with the correct value.


5. The text of the "Save configuration" button  changes to "Update Configuration".
